# 前端開發服務器手動啟動指南

## 步驟 1: 打開終端
- 按 `Win + R`，輸入 `cmd` 或 `powershell`，按 Enter
- 或者直接在項目文件夾中，按住 `Shift` 右鍵，選擇「在此處打開 PowerShell 窗口」

## 步驟 2: 進入前端目錄
```bash
cd C:\hbgm001\frontend
```

## 步驟 3: 檢查依賴
```bash
# 檢查 node_modules 是否存在
dir node_modules

# 如果不存在，安裝依賴
npm install
```

## 步驟 4: 啟動開發服務器
```bash
npm run dev
```

## 步驟 5: 等待服務器啟動
你應該會看到類似這樣的輸出：
```
VITE v5.x.x  ready in xxx ms

➜  Local:   http://localhost:3001/
➜  Network: use --host to expose
```

## 步驟 6: 打開瀏覽器測試
1. 看到 "ready" 消息後，打開瀏覽器
2. 訪問：`http://localhost:3001`
3. 應該能看到應用界面

## 步驟 7: 測試新功能
1. 導航到「發紅包」頁面
2. 點擊「選擇群組」按鈕
3. 測試搜索功能：
   - 輸入群組名稱
   - 輸入群鏈接（如 `t.me/xxx`）
   - 輸入用戶名（如 `@username`）
   - 確認自動搜索並顯示結果

## 常見問題排查

### 問題 1: 端口被占用
如果看到端口被占用的錯誤：
```bash
# 查找占用端口的進程
netstat -ano | findstr :3001

# 終止進程（替換 PID 為實際的進程 ID）
taskkill /F /PID <PID>
```

### 問題 2: 依賴安裝失敗
```bash
# 清理並重新安裝
rmdir /s /q node_modules
del package-lock.json
npm install
```

### 問題 3: 編譯錯誤
如果看到 TypeScript 錯誤：
```bash
# 檢查編譯錯誤
npx tsc --noEmit
```

### 問題 4: 服務器啟動但無法訪問
- 檢查防火牆設置
- 確認瀏覽器地址欄輸入的是：`http://localhost:3001`（不是 `https://`）

## 停止服務器
在運行服務器的終端窗口中按 `Ctrl + C`
